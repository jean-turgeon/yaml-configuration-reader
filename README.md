# yaml-configuration-reader

A small demo on how to read a Yaml config file and make it available as a key, value map.


## Repository content
.
├── Cargo.lock
├── Cargo.toml
├── README.md
└── src
    ├── configs.yaml
    ├── configuration.rs
    └── main.rs


## Build
```bash
cargo build
```

### Run
```bash
cargo run
```
